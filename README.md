# nuts3745.dev

A dependency-free personal website built with vanilla HTML and CSS.

## Features

- Static single-page site
- Responsive layout
- Social links for Bluesky, GitHub, Scrapbox, and Laboratory
- No npm dependencies, build step, or JavaScript runtime

## Project Structure

- `index.html` - The complete page markup
- `styles.css` - Site styling
- `public/favicon.svg` - Favicon asset

## Local Preview

Open `index.html` directly in a browser, or serve the directory with any static file server.

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).

## Deployment

Deploy the repository root as static files. No install or build command is required.
